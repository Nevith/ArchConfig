# arch-dotfiles
Dotfiles for arch linux 

![Screenshot](https://github.com/planeer/arch-dotfiles/blob/master/screenshots/screenshot.jpg)

### Symlinks
* ~/ & /root/:
    * .Xresources
* ~/.config/:
    * i3
    * ranger
    * dunst
    * polybar
    * background.png
    * login_background.png
* /etc/:
    * lightdm
